# 📉 Customer Churn Analysis

## Overview

This project analyzes customer behavior to identify factors contributing to customer churn.

The analysis uses Python, Pandas, NumPy, Matplotlib, and Seaborn to explore customer demographics, subscription details, and service usage patterns.

---

## Objectives

- Analyze customer churn patterns
- Identify high-risk customers
- Compare churn across contract types
- Study impact of monthly charges
- Generate business recommendations

---

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
