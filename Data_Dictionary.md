# Data Dictionary

| Column | Data Type | Description |
|---------|-----------|-------------|
| Customer_ID | VARCHAR | Unique customer identifier |
| Gender | VARCHAR | Gender of the customer |
| Age | INT | Customer age |
| Tenure | INT | Number of months the customer has stayed |
| Monthly_Charges | DECIMAL | Monthly subscription charges |
| Total_Charges | DECIMAL | Total amount paid by the customer |
| Contract | VARCHAR | Contract type (Month-to-Month, One Year, Two Year) |
| Internet_Service | VARCHAR | Internet service type |
| Payment_Method | VARCHAR | Payment method used |
| Tech_Support | VARCHAR | Whether technical support is subscribed |
| Online_Backup | VARCHAR | Whether online backup service is subscribed |
| Streaming_TV | VARCHAR | Whether streaming TV service is subscribed |
| Churn | VARCHAR | Indicates whether the customer left the company (Yes/No) |
